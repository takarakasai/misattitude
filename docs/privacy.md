# Misattitude プライバシーポリシー

最終更新日: 2026-05-06

## 1. はじめに

本プライバシーポリシー（以下「本ポリシー」）は、開発者 Takara Kasai（以下「開発者」）が
提供する Android アプリケーション「Misattitude」（以下「本アプリ」）における、
利用者の情報の取扱いについて定めるものです。本アプリをご利用いただくことにより、
利用者は本ポリシーに同意したものとみなします。

## 2. 本アプリの概要

本アプリは、3 次元剛体の姿勢（attitude / rotation）をクォータニオン・回転行列・
オイラー角の 3 つの数学表現で可視化する教育用ツールです。すべての処理は端末内で
完結し、外部サーバーとの通信は行いません。

## 3. 取得する情報

本アプリは、以下の情報を**いずれも取得しません**:

- 氏名・メールアドレス・電話番号などの個人を特定できる情報
- 位置情報
- カメラ・マイク・連絡先・ストレージ等のデバイス機能
- アカウント情報・認証情報
- 端末の固有識別子（IMEI、広告 ID 等）
- 利用ログ・操作履歴・クラッシュレポート

## 4. 第三者サービス

本アプリは、現時点では情報を取得・送信する第三者サービスを利用していません。

## 5. 必要な権限（Permission）

本アプリは、Android のいかなる実行時権限（runtime permission）も要求しません。

## 6. 子どもの安全

本アプリは特定の年齢層を対象としていませんが、教育目的のため一般利用者向けに
設計されています。13 歳未満の子どもからの個人情報を意図的に収集することはありません。

## 7. ポリシーの変更

将来、本アプリの機能拡張により情報の取扱いに変更が生じる場合、本ポリシーを更新し、
本アプリ内またはアプリストアの説明欄でお知らせします。重要な変更がある場合は、
更新日を本ページ上部に明記します。

## English Version (Summary)

### Misattitude Privacy Policy

Last updated: 2026-05-06

This application ("Misattitude") is an educational tool for visualizing 3D
rigid-body attitude in three mathematical representations (quaternion,
rotation matrix, and Euler angles). All processing is performed locally on
the user's device.

**The application does NOT collect, store, or transmit any of the following:**

- Personally identifiable information (name, email, phone number, etc.)
- Location data
- Device sensors (camera, microphone) or contacts
- Authentication credentials
- Device identifiers (IMEI, advertising ID, etc.)
- Usage logs, behavioral data, or crash reports

The application requires **no Android runtime permissions** and uses **no
third-party services** that collect user data.

For inquiries, please contact: `kohya.th@gmail.com`
Repository: https://github.com/takarakasai/misattitude
