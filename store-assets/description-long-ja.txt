Misattitude は、3次元の「姿勢（向き）」を表す3つの数学表現 ―― クォータニオン・回転行列・オイラー角 ―― を、同じ回転として同時に可視化する教育用ツールです。回転を扱うすべての分野（ロボティクス、SLAM、コンピュータグラフィックス、航空宇宙、ゲーム開発、IMU/AR）で、初学者がつまずきやすい「同じ回転なのに、表記がいくつもある」という壁を、目で見て直感的に理解できるよう設計しました。

■ できること

・3表現の同期表示
  どのパネル（クォータニオン / 回転行列 / オイラー角）で値を変更しても、内部のキューブと他の2表現が即座に追従します。「同じ姿勢を3通りに眺める」体験を、操作のたびに繰り返せます。

・12通りのオイラー順序を完全網羅
  Tait-Bryan型6種（XYZ・XZY・YXZ・YZX・ZXY・ZYX）と、Proper Euler型6種（XYX・XZX・YXY・YZY・ZXZ・ZYZ）の合計12通りを切り替え可能。ロボット工学のZYX、CGのXYZ、ジャイロスコープ系のZXZなど、分野ごとの慣習をその場で再現できます。

・内在 / 外在の切り替え
  各回転を「直前の姿勢から見たボディ軸」に対して行う内在（intrinsic）と、「動かない世界軸」に対して行う外在（extrinsic）を、スイッチ一つで切り替え。表記は変わっても結果は同じになる、という回転の対称性を体感できます。

・ジンバルロックの可視化
  オイラー角が特異点に近づくと、画面上に警告チップとハイライトが表示され、自由度が失われていることを直感的に確認できます。教科書では数式でしか触れられない「ジンバルロック」が、実際にスライダーを動かす中で立ち上がる様子を見ることができます。

・段階分解（Step decomposition）の表示
  「Show step axes」を有効化すると、オイラー角の3要素を順に1回ずつ適用したときの中間姿勢のボディ軸が、フェードアウトした補助線として描画されます。ZYX intrinsic の3つの回転を視覚的に追跡できる教材としてご活用ください。

・Slerp と Euler-LERP の比較再生
  開始姿勢と終了姿勢をキャプチャし、3つの補間モード（Slerp / Euler-LERP / Step）で滑らかに再生・比較できます。「比較ゴースト」を有効化すると、もう一方の補間軌道がワイヤフレームキューブで重畳表示され、Slerp が SO(3) 上の測地線である一方、Euler-LERP は最短経路を取らないことが目で分かります。

・3D描画は Google Filament による物理ベースレンダリング
  Filament の高品質なレンダリングで、半透明キューブ・カラーラベル・ボディ軸・ワールド軸が見やすく表現されます。タッチ操作（ドラッグ・ピンチ・ダブルタップでリセット）でカメラを自由に操作できます。

・世界座標系規約の切り替え
  Y-up 右手系（OpenGL / Unity 系）と Z-up 右手系（ROS / 航空宇宙系）をプリセット切替。同じクォータニオンを異なる世界規約で眺めることで、「規約の違いは姿勢を変えない」という不変量が体感できます。

・3種のボディ形状
  半透明リファレンスキューブ、Linuxマスコット風ペンギン、ロボット風カプセル ―― から選択。形がついた物体だと「上下」「前後」が直感的に分かるため、ロボット姿勢の入門に最適です。

■ こんな方におすすめ

・大学生：制御工学・ロボット工学・コンピュータグラフィックスの講義で、回転表現の理解を補強したい方
・エンジニア：実装するコードで「どのEuler慣習か」「Hamilton か JPL か」「intrinsic か extrinsic か」に頭を悩ませる方
・教員・指導者：講義スライドだけでは伝わりにくい「ジンバルロック」「Slerp vs Linear」を、ライブで見せたい方
・趣味で勉強中の方：四元数を初めて学ぶ際、「数式が理解できても、何をしているか感覚的に分からない」状態を抜け出したい方

・OpenGL ES 3.x 対応の GPU を搭載した実機を推奨（Filament の3D描画を快適に表示するため）

回転表現の数学は一見シンプルでも、複数の慣習・規約が交錯するため、現場で繰り返し迷う領域です。本アプリが、数式と直感の橋渡しになれば幸いです。
